Project Description: Time Tracking & Productivity Analytics Chrome Extension

The Time Tracking & Productivity Analytics Chrome Extension is a full-stack web-based application designed to monitor user browsing behavior and provide meaningful insights into productivity patterns. This project combines a browser extension, a backend server, and a web dashboard to create a complete system for tracking and analyzing time spent on different websites.

At its core, the Chrome extension runs in the background and continuously monitors the active browser tabs. Whenever a user switches tabs or opens a new website, the extension records the time spent on the previous site. It extracts the domain name and calculates the duration of activity. This data is then categorized into three types: productive, unproductive, and neutral, based on predefined lists of websites. For example, platforms like coding websites are marked as productive, while social media sites are considered unproductive.

The collected data is sent to a backend server built using Node.js and Express. The backend acts as a central system for storing and managing user activity data. It exposes APIs that allow the extension to send tracking data and the dashboard to retrieve analytics. collection and data visualization.
